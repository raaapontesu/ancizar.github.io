Ancízar 8.0
Pangrama comparativo de glifos para Ancizar Sans
Unicode: Basic Latin (0000-007F)

Esta página está diseñada para poner a prueba las características de Ancizar Sans respecto al hinting, kerning y otras cualidades tipográficas que deben ser tenidas en cuenta. Dos de las mejores fuentes tipográficas tipo sans jamás creadas son Andika y Noto Sans; serán usadas como referencia. Es importante tener en cuenta que cada una tiene sus fortalezas, pero también sus debilidades. Aquí se notará el desempeño de Ancizar Sans conforme se vaya avanzando en la creación de glifos.

Cuando se escribió este documento, se empleó la versión 7.0 de Ancizar, del 2 de abril del 2016. Para comenzar a incorporar nuevos glifos, se deben exportar las familias de interés en formato WOFF2, con la denominación estándar contemplada en los nombres de archivo, y almacenarlas en el directorio Fuentes/Ancizar Sans/, el cual está diseñado para albergarlas.
